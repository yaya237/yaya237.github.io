Mes Réalisations

Projet 01 - Chatbot Intelligent

Développement d’un assistant virtuel évolutif avec frontend React et backend Node.js/Express. Intégration d’un moteur NLP pour comprendre les intentions, système de logging et analytics pour suivre les interactions. Déploiement sur Docker/Kubernetes avec CI/CD via GitHub Actions. Version cloud sécurisée sur AWS et Azure.

Projet 02 - SIEM OpenSearch

Mise en place d’une solution de sécurité pour monitorer un cloud privé avec OpenSearch. Collecte et centralisation des logs, mise en place de règles d’alerte avec Elastalert pour anomalies et comportements suspects. Surveillance en temps réel et automatisation de la gestion des logs.

Projet 03 - Conception d'un Data-center

Création d’une plateforme cloud interne (type Azure/AWS) pour l’école. Configuration réseau complète, pare-feu OPNSense, monitoring avec Zabbix et Grafana. Automatisation via Ansible et Vagrant, stratégies de sauvegarde et sécurité renforcée.

Projet 04 - InfraLogiciel (Dev&Ops)

Conception, mise en ligne et sécurisation d’un site web complet avec 5 étudiants. Technologies: HTML, CSS, JS, Node.js, Vagrant, Shell, Git. Renforcement des compétences DevOps, sécurité et workflow collaboratif.

PFE - Cloud Privé OpenNebula

Mise en place d’une infrastructure cloud privée avec OpenNebula. Déploiement et gestion de machines virtuelles, configuration réseaux virtuels et orchestration des ressources. Automatisation via scripts Shell et Vagrant, monitoring avancé avec Zabbix. Acquisition d’une solide expérience sur les environnements cloud privés.

POC - Keycloak

Réalisation d’un Proof of Concept pour centraliser l’authentification et la gestion des utilisateurs. Configuration de realms, clients, rôles et flux sécurisés. Intégration testée sur différentes applications pour améliorer sécurité et conformité.
